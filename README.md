# Console-Square-Puzzle


A procedural shape-fitting puzzle game built entirely in C# for the console. I developed this project as part of my Project-Based Learning (PBL) coursework in the Computer Engineering program at Dokuz Eylül University. It demonstrates a strong understanding of complex 2D array manipulation, matrix transformations, and interactive console UI.

## Features
* **Procedural Piece Generation:** Custom algorithms to normalize, rotate, and reverse matrix shapes.
* **Collision Detection:** Ensures shapes fit properly within the grid boundaries and do not overlap with existing pieces.
* **Dynamic Scoring:** Calculates "regularity" and assigns a round score based on the mathematical perfection of the piece placement.
* **Interactive UI:** Fully playable directly in the console using real-time keyboard inputs.

## Controls
* **Arrow Keys:** Move the selected piece around the grid.
* **R:** Rotate the piece 90 degrees.
* **M:** Mirror (reverse) the piece.
* **Enter:** Place the piece on the grid.
* **A-T (Letters):** Select a specific piece from your generated inventory to place.
